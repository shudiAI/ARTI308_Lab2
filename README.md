# ARTI308_Lab2
Dataset Name: BMW Used Car Dataset

Source: Kaggle

Dataset Format: CSV

Number of Records: 10,781

Number of Features: 9

Machine Learning Problem Type: Supervised Learning – Regression

Target Variable: Price

Prediction Goal:
Predicting the price of a used BMW car based on its features such as model, year, mileage, transmission type, fuel type, engine size, fuel efficiency (mpg), and tax.
